# 竞品体验分析

从用户体验视角系统性对比竞品，识别体验差距、行业最佳实践和差异化机会，为体验优化提供基准和方向。

## 触发条件

- 需要从体验视角对比竞品
- 需要发现体验差距和差异化机会
- 需要了解行业体验最佳实践
- 产品需要重新定位体验策略
- 需要为体验优化设定基准目标

## 核心能力

### 1. 竞品选择

根据分析目标选择合适的竞品层次：

| 层次 | 定义 | 分析重点 |
|------|------|---------|
| 直接竞品 | 同一品类、同一目标用户 | 功能体验对比、核心流程对比 |
| 间接竞品 | 不同品类但解决同一需求 | 替代方案体验、需求满足方式 |
| 参照标杆 | 体验最佳的行业标杆 | 最佳实践学习、标杆设定 |
| 跨界参照 | 跨行业的体验创新 | 体验创新启发、思维突破 |

竞品数量建议 3-5 个，过多则分析深度不足。

### 2. 体验分析维度

从以下维度系统对比竞品体验：

**第一印象体验**
- 首次打开的认知成本
- 新手引导的友好程度
- 核心价值的传达速度

**核心流程体验**
- 核心任务的完成路径长度
- 每步的操作复杂度
- 流程中的容错与恢复
- 完成后的反馈与闭环

**信息架构体验**
- 信息层级是否清晰
- 导航是否直觉
- 搜索效率如何
- 内容组织逻辑

**交互细节体验**
- 操作反馈是否即时
- 手势/快捷操作支持
- 动效是否辅助理解
- 错误提示是否友好

**情感体验**
- 视觉风格的情感传达
- 品牌个性的一致性
- 惊喜时刻的设计
- 情绪曲线的走势

**服务体验**（如适用）
- 帮助渠道的可达性
- 问题解决的效率
- 自助服务的完善度
- 人工服务的衔接

### 3. 分析方法

**体验矩阵打分法**
- 为每个维度设定 1-5 分评分标准
- 对自身和每个竞品打分
- 计算总分和各维度得分
- 识别优势/劣势/持平维度

**关键流程对比法**
- 选取 3-5 个核心流程
- 逐步骤对比竞品的处理方式
- 标注体验差异点和亮点
- 提炼可借鉴的做法

**体验差距地图**
- 以"体验重要性"和"当前表现"为两轴
- 将各维度定位到地图上
- 识别高重要-低表现的体验短板
- 识别高重要-高表现的优势领域

### 4. 差异化机会识别

从竞品分析的空白地带发现差异化机会：
- 竞品都做但体验差的领域 → 体验超越机会
- 竞品都未做的领域 → 体验创新机会
- 竞品各有优劣的领域 → 取长补短机会
- 行业标杆做法 → 快速学习机会
