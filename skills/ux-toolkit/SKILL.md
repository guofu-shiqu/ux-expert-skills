# UX Toolkit — 用户体验工具包

为 UX 分析提供结构化方法论框架和实用模板。

## 包含内容

### references/ 目录

| 文件 | 内容 |
|------|------|
| `heart-model.md` | HEART 模型完整定义、指标分解、评分标准与计算公式 |
| `journey-map-template.md` | 用户体验地图模板，包含阶段、触点、情绪曲线、痛点与机会点 |
| `heuristic-evaluation.md` | Nielsen 十大可用性启发式原则检查清单 |
| `usability-test-script.md` | 可用性测试标准脚本模板（含任务设计、观察要点、SUS 量表） |
| `hci-principles.md` | 核心 HCI 交互法则速查（费茨定律、希克定律、格式塔原则等） |

## 使用方式

在对话中提及对应方法时，读取 references/ 下对应文件获取详细定义和模板，确保输出内容符合业界标准框架。
